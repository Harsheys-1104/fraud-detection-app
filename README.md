 💳 Credit Card Fraud Detection System
 
 🚀 Overview
This project is a machine learning-based system designed to detect fraudulent credit card transactions using imbalanced financial data.

 🧠 Features
- Fraud detection using ML models
- Handles imbalanced datasets
- Real-time prediction via web app
- Interactive UI using Streamlit

 🛠️ Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
- Streamlit

 ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
